## Install
### Post INSTALL
- git
- [Alacritty](https://alacritty.org/index.html)
### MacOS/Linux
```bash
git clone https://github.com/kanrin/my-alacritty-config.git $HOME/.config/alacritty
```
### Windows
```cmd
git clone https://github.com/kanrin/my-alacritty-config.git %APPDATA%\alacritty
```
